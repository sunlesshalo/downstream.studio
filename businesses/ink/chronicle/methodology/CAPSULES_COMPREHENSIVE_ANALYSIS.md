## KEY PRODUCTION INSIGHTS

### 1. ATTENTION CAPTURE (Opening)

**Rule:** First scene MUST have HIGH attention element:
- Option A: Saturated colorful subject (red bird)
- Option B: Luminous/glowing element (amber orb)
- BOTH against contrasting background (cold/dark)
- Subject should be ANIMATED or GLOWING (not static silhouette)

### 2. MOTION DISTRIBUTION

**Rule:** Most scenes are HOLDS. Motion is the exception.
- 1-2 scenes: HIGH motion (opening, one mid-point)
- 1 scene: PARALLAX (complexity/richness)
- 1 scene: STATE CHANGE (emotional turning point)
- 5-7 scenes: ATMOSPHERIC HOLD (majority)
- 1 scene: STATIC (ending/resolution)

### 3. COLOR TEMPERATURE

**Rule:** Alternate warm/cold, with intentional arc:
- Opening: Warm element visible (even if background cold)
- Middle: Alternate based on content
- Ending: Can be either (matches emotional resolution)

### 4. PHOTOREALISM USAGE

**Rule:** Reserve photorealism for intimate human moments:
- Connection scenes
- Emotional intimacy
- Use stylized for symbolic/conceptual content

### 5. BOOKEND STRATEGY

**Rule:** Opening and closing should have clear relationship:
- ECHO: Same visual element, transformed meaning
- CONTRAST: Opposite visual elements, related meaning
- INVERSION: Full flip (young→old, digital→analog)

### 6. SCENE DURATION

| Function | Duration |
|----------|----------|
| Hook | 1500-2000px |
| Standard | 1500-2000px |
| Rest point | 2000-2700px |
| Resolution | 2000-2500px |

### 7. TEXT-VISUAL RELATIONSHIP

**Rule:** Default to SYMBOLIC (visuals represent feeling, not literal)
- Never illustrate text literally
- Visuals accompany emotional register
- Sync rare moments where visual metaphor = text metaphor
